# Code of Conduct

This project follows a simple standard: be respectful, constructive, and professional.

## Expected behavior

- Be respectful in communication
- Assume good intent, but discuss technical concerns directly
- Give actionable feedback
- Keep disagreements technical, not personal

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Doxxing or disclosure of private information
- Trolling, intimidation, or hostile conduct
- Repeated disruptive behavior after warnings

## Enforcement

Maintainers may remove content or block participation for behavior that harms the project.

## Scope

This applies to project spaces (issues, PRs, discussions, chat) and external spaces when representing the project.

## Reporting

Report incidents privately to maintainers via GitHub or project security/reporting channels.
